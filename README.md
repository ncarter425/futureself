🛡️ FutureSelf: The Freedom Impact Dashboard

**Most budgeting apps tell you what you spent. FutureSelf shows you the life you traded for it.**

FutureSelf is a visceral financial auditing tool designed to visualize the compounding consequence of daily habits. By converting monthly expenses into "Years of Labor," the app provides a psychological gut-check that standard spreadsheets can't match.

## 🚀 The Vision
- **Not Just Math:** We turn abstract dollar amounts into "Years of Freedom."
- **Opportunity Cost:** See the $180,000 retirement impact of a $200/month coffee habit.
- **Decision Engine:** Use the "Kill a Habit" toggle to see your retirement age drop in real-time.

## 🛠️ Tech Stack
- **Frontend:** [Streamlit](https://streamlit.io) (Python-based Web Framework)
- **Visuals:** [Plotly](https://plotly.com) for interactive wealth erosion charts.
- **Logic:** Financial modeling based on the Future Value of an Annuity formula.
- **Database:** [Supabase](https://supabase.com) (PostgreSQL) for persistent data saving on mobile.

## 📦 Local Installation
To run this project locally:

1. Clone the repository:
   ```bash
   git clone https://github.com](https://github.com/ncarter425/futureself.git
   ```
2. Install dependencies:
   ```bash
   pip install streamlit pandas numpy plotly st-supabase-connection
   ```
3. Run the application:
   ```bash
   streamlit run app.py
   ```

## 📱 Mobile Setup
To use this as a phone app:
1. Open the deployed URL in your phone's browser.
2. Tap **'Share'** (iOS) or the **'Menu'** (Android).
3. Select **'Add to Home Screen'**.
